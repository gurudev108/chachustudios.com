# chachustudios.com

Welcome to the Chachu Studios project! This is the landing page for Chachu Studios, designed to introduce our mission and services. Built with Hugo, this project aims to blend simplicity, elegance, and ease of deployment.

---

## About Chachu Studios

Chachu Studios is a unique space where children thrive, parents find peace of mind, and ancient wisdom meets modern needs. Our mission is to create a holistic, nurturing, and creative environment for kids and parents alike.

---

## Features

- **Yoga and Meditation**: Introduce children to yoga philosophy, asanas, and calming meditation techniques.
- **Nature and Animals**: Interaction with animals (dogs, cats, horses) and nature hikes to foster compassion and joy.
- **Creative Arts**: Recording studio for music and art, allowing children to explore their creativity.
- **Daycare and Learning**: Professional care combined with reading programs and interactive activities.
- **For Parents**: Gym facilities for mothers, with professional childcare support.

## Folder Structure

Below is the folder structure for the project:

```plaintext
Chachu-Studios/
├── archetypes/        # Templates for new content types
├── assets/            # Source files for SCSS, JS, and images
├── content/           # Markdown files for website content
│   ├── _index.md      # Homepage content (TBD)
│   └── blog/          # Blog posts
├── data/              # Data files (e.g., JSON, YAML, or TOML)
├── i18n/              # Translations for multilingual support
├── layouts/           # Custom HTML and template overrides
├── public/            # Generated static files after a Hugo build
├── static/            # Static files (images, CSS, JS) served as-is
├── themes/            # Themes for Hugo
├── config.toml        # Hugo configuration file
├── README.md          # Project documentation
└── LICENSE.txt        # License information
```
